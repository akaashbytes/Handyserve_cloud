# HandyServe Cloud

HandyServe is a cloud-based microservices application deployed on AWS EC2.

## Tech Stack

### Frontend
- React
- Vite

### Backend
- Java 17
- Spring Boot
- Maven
- Spring Cloud Gateway
- JWT Authentication

### Microservices
- API Gateway
- Authentication Service
- Booking Service
- Communication Service
- Common Module

## Deployment
AWS EC2 (Amazon Linux 2023)
